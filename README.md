# qgis-models-plugin

[QGIS](https://github.com/qgis/QGIS) plugin that allows you to use following math models:
* [Central place theory](https://en.wikipedia.org/wiki/Central_place_theory)
* [Regression analysis](https://en.wikipedia.org/wiki/Regression_analysis)
* [Reilly's law of retail gravitation](https://en.wikipedia.org/wiki/Reilly%27s_law_of_retail_gravitation)
* [Модель Тюнена](https://ru.wikipedia.org/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%A2%D1%8E%D0%BD%D0%B5%D0%BD%D0%B0) — [(см. подробнее)](http://gallery.economicus.ru/cgi-bin/frame_rightn.pl?type=school&links=./school/firm/lectures/firm_l2.txt&name=firm&img=lectures.jpg)
* [Теория организации экономического пространства](https://ru.wikipedia.org/wiki/%D0%9B%D1%91%D1%88,_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82#Теория_организации_экономического_пространства:~:text=%5B7%5D.-,%D0%A2%D0%B5%D0%BE%D1%80%D0%B8%D1%8F%20%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D0%B8%20%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B3%D0%BE%20%D0%BF%D1%80%D0%BE%D1%81%D1%82%D1%80%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%B0,-%5B%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D1%8C%20%7C)

The plugin uses the official [QIS Python API](https://qgis.org/pyqgis/master/)
